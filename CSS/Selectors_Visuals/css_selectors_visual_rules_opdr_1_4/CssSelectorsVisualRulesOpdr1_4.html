Opdracht 1. Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. External CSS

      2. Met het external style sheet, kan je de stijl van een volledige website veranderen met slechts 1 file! Elke HTML pagina moet een referentie bevatten naar het externe stylesheet bestand in het <link> element, in de head sectie. Een externe style sheet kan geschreven worden in elke text editor en moet opgeslagen worden met een .css extentie. Het externe css bestand mag geen HTML tags bevatten.

  2. Internal CSS

      3. Het internal style sheet mag gebruikt worden als slechts een enkele HTML pagina een unieke stijl heeft. De internal stijl is gedefinieerd binnen het <style> element, binnen-in de head section.

  3. Inline CSS

      4. Een inline style mag gebruikt worden om een unieke stijl toe te passen voor een enkel element. Om inline stijlen toe te passen, voeg je de style attribute toe aan het relevante element. De style attrribute kan elke CSS property bevatten.

Opdracht 2. CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Het basis doel van de CCS taal is om een browser bepaalde elementen van een pagina in te kleuren en te decoreren. Met verschillende kleuren, posities of decoraties. Het CSS syntax reflecteerd dit doel en zijn basis onderdelen zijn: De property wat een identifier is, dat is een naam die door mensen te lezen is die definieerd wat zijn functie is. En de value welke beschrijft hoe die functie moet worden toegepast door de computer. Elke property heeft een set van gevalideerde values, gedefinieerd door abc taal, met ook een sematic betekenis, geimplementeerd door de browser.

Opdracht 3. Er zijn verschillende selectors, bijvoorbeeld;

  • type selectors
  • descendant selectors
  • class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

type selectors:

The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.


descendant selectors:

The descendant selector matches all elements that are descendants of a specified element. The first simple selector within this selector represents the ancestor element—a structurally superior element, such as a parent element, or the parent of a parent element, and so on. The second simple selector represents the descendant element we’re trying to match. The combinator we use in a descendant selector is a whitespace character: a space, horizontal tab, carriage return, line feed, or form feed. Since whitespace characters are allowed around all combinators, you can include more than one whitespace character between the simple selectors in a descendant selector.


class selectors:

The CSS class selector matches elements based on the contents of their class attribute.
