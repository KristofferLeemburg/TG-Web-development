Opdracht 1. Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. External CSS

      2. Met het external style sheet, kan je de stijl van een volledige website veranderen met slechts 1 file! Elke HTML pagina moet een referentie bevatten naar het externe stylesheet bestand in het <link> element, in de head sectie. Een externe style sheet kan geschreven worden in elke text editor en moet opgeslagen worden met een .css extentie. Het externe css bestand mag geen HTML tags bevatten. Er zijn geen style tags in het html document. Het grote voordeel van een extern css bestand is dat je het meerdere keren kan gebruiken, Het .css bestand is geen deel van een specifiek html document maar elke pagina kan het gebruiken.

    Voorbeeld <html>
    <head>
    <link rel="stylesheet" type="text/css" href="mystyle.css"></link>
    </head>

  2. Internal CSS

      3. Het internal style sheet mag gebruikt worden als slechts een enkele HTML pagina een unieke stijl heeft. De internal stijl is gedefinieerd binnen het <style> element, binnen-in de head section. Door het style element te gebruiken kan je geen gebruik maken van een extern stylesheet.

    Voorbeeld <head>
    <style>
    h1 {
    color: red;
    margin-left: 20px
    }
    </style>
    </head>

  3. Inline CSS

      4. Een inline style mag gebruikt worden om een unieke stijl toe te passen voor een enkel element. Om inline stijlen toe te passen, voeg je de style attribute toe aan het relevante element. De style attrribute kan elke CSS property bevatten.

    Voorbeeld <h1 style="color:blue;text-align:center;">This is a heading</h1>

Opdracht 2. CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Het basis doel van de CCS taal is om een browser bepaalde elementen van een pagina in te kleuren en te decoreren. Met verschillende kleuren, posities of decoraties. Het CSS syntax reflecteerd dit doel en zijn basis onderdelen zijn: De property wat een identifier is, dat is een naam die door mensen te lezen is die definieerd wat zijn functie is. En de value welke beschrijft hoe die functie moet worden toegepast door de computer. Elke property heeft een set van gevalideerde values, gedefinieerd door abc taal, met ook een sematic betekenis, geimplementeerd door de browser.

Voorbeeld Background-color:red dan is background-color de property en red de value en de colon scheid de twee

Opdracht 3. Er zijn verschillende selectors, bijvoorbeeld;

  1. type selectors
  2. descendant selectors
  3. class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  1. type selectors:

      2. De type selector komt overeen met elementen op nodename. Met andere woorden, het selecteert alle elementen van het gegeven type binnen een document.

  2. descendant selectors:

      3. De descendant selector komt overeen met alle elementen die afstammelingen zijn van een opgegeven element. De eerste eenvoudige selector binnen deze selector vertegenwoordigt het voorouderelement - een structureel superieur element, zoals een parent element, of het bovenliggende element van een bovenliggend element, enzovoort. De tweede eenvoudige selector vertegenwoordigt de afstammeling element dat we proberen te matchen. De combinator die we gebruiken in een afstammende selector is een witruimteteken: een spatie, horizontale tab, return, line feed of form feed. Aangezien witruimte-tekens zijn toegestaan ​​rond alle combinatoren, kunt u meer dan één witruimteteken tussen de eenvoudige selectors in een afstammende selector.


  3. class selectors:

      4. De class selector komt overeen met elementen op basis van de inhoud van hun class attribuut.
