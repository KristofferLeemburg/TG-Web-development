CSS Selectors

CSS selectors worden gebruikt om bepaalde HTML elementen te vinden of te selecteren die je wilt stylen.

We kunnen de CSS selectors in vijf catogorieren plaatsen:

  • Simple selectors (selecteerd elementen op naam, id, class)
  • Combinator selectors (selecteerd elementen op een specifieke relatie tussen elkaar)
  • Pseudo-class selectors (selecteerd elementen op een bepaalde staat)
  • Attribute selectors (selecteerd elementen op een attribuut of attribuut waarde)



Selector Voorbeeld Voorbeeld omschrijving
.class .intro Selecteerd alle elementen met de class="intro"
.class1.class2 .name1.name2 Selecteerd alle elementen met beide name1 and name2 set binnenin de class attribuut
.class1 .class2 .name1 .name2 Selecteerd alle elementen met name2 die een descendant zijn van een element met name1
#id #firstname Selecteerd het element met id="firstname"
* * Selecteerd alle elementen
element p Selecteerd alle <p> elementen
element.class p.intro Selecteerd alle <p> elementen met class="intro"
element.class p.intro Selecteerd alle <p> elementen met class="intro"
element.class p.intro Selecteerd alle <p> elementen met class="intro"
element element div p Selecteerd alle <p> elementen binnenin <div> elementen
element>element div > p Selecteerd alle <p> elementen waar de parent een <div> element is
element+element div + p Selecteerd alle <p> elementen die geplaatst zijn direct na <div> elementen
element1~element2 p ~ ul Selecteerd alle <ul> elementen die voorafgegaan door een <p> elementen
[attribute] [target] Selecteerd alle elementen met een target attribuut
[attribute=value] [target=_blank] Selecteerd alle elementen met target="_blank"
[attribute~=value] [title~=flower] Selecteerd alle elementen met een titel attribuut die het woord "flower" bevatten
[attribute|=value] [lang|=en] Selecteerd alle elementen met een lang attribuut waarde startend met "en"
[attribute^=value] a[href^="https"] Selecteerd alle <a> elementen welke href attribuut waarde begins met "https"
[attribute$=value] a[href$=".pdf"] Selecteerd elk <a> element whose href attribuut waarde ends met ".pdf"
[attribute*=value] a[href*="w3schools"] Selecteerd elk <a> element whose href attribuut waarde contains de substring "w3schools"
