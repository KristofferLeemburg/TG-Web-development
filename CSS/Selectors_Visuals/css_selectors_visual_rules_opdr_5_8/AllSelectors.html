CSS Selectors

CSS selectors worden gebruikt om bepaalde HTML elementen te vinden of te selecteren die je wilt stylen.

We kunnen de CSS selectors in vijf catogorieren plaatsen:

  • Simple selectors (selecteerd elementen op naam, id, class)
  • Combinator selectors (selecteerd elementen op een specifieke relatie tussen elkaar)
  • Pseudo-class selectors (selecteerd elementen op een bepaalde staat)
  • Attribute selectors (selecteerd elementen op een attribute of attribute value)
