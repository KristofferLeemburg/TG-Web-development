Opdracht 1. Wat zijn de eigenschappen van block boxes en inline boxes?

In CSS hebben we ongeveer twee soorten boxen. Block boxes en inline boxes. Deze karakteristieken refereren naar hoe de box zich gedraagd in termen van een vloeiende pagina, en in relatie tot andere boxes op de pagina:
Als een box is gedefininieerd als een block, zal het zich op de volgende manier gedragen:

  • De box zal beginnen op een nieuwe regel.
  • De box zal The box will extend in the inline direction to fill the space available in its container. In most cases this means that the box will become as wide as its container, filling up 100% of the space available.
  • De width en height properties zullen worden behouden.
  • Padding, margin and border zal ervoor zorgen dat andere elementen weggeduwd worden van de box.

Alleen wanneer we ervoor kiezen om de display type te veranderen naar inline, dan zullen elements als headings (e.g. <h1>) en <p> allemaal de block genruiken als hun display type als standaard.
If a box has an outer display type of inline, then

  • The box will not break onto a new line.
  • The width and height properties will not apply.
  • Vertical padding, margins, and borders will apply but will not cause other inline boxes to move away from the box.
  • Horizontal padding, margins, and borders will apply and will cause other inline boxes to move away from the box.
  • The element, used for links, <span>, <em> and <strong> are all examples of elements that will display inline by default.

Het type box dat op een element wordt toegepast, wordt bepaald door waarden voor weergave-eigenschappen zoals block en inline, en heeft betrekking op de uiterlijke waarde van het display.
