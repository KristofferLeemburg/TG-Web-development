Opdracht 4. Leg uit wat de volgende termen betekenen: content, padding, margin en border.

Content

CSS heeft een property genoemd content. Het kan alleen gebruikt worden met de pseudo elementen :after and :before. Het is geschreven als een pseudo-selector (met de dubbele punt), maar het wordt een pseudo-element genoemd omdat het niet echt is alles selecteren dat bestaat op de pagina maar iets nieuws aan de pagina toevoegen. Dit is hoe het eruit ziet:

Voorbeeld .email-address:before{
content: "Email address: ";
}

Padding

De padding property in CSS definieerd het binnenste deel van het boxmodel, waardoor ruimte ontstaat rond een inhoud van het element, binnen gedefinieerde marges en / of randen. Opvulwaarden worden ingesteld met lengtes of percentages, en kan geen negatieve waarden hebben. De standaardwaarde voor alle opvullingseigenschappen is 0.

Voorbeeld .box {
padding: 0 1.5em 0 1.5em;
}

Margin

Voorbeeld .box {
margin: 0 3em 0 3em;
}

Border

De eigenschap border is een afgekorte syntax in CSS die meerdere waarden accepteert voor het tekenen van een lijn rond het element waarop het wordt toegepast.

Voorbeeld border: 3px solid red;
height: 200px;
width: 200px;
}
