10. Om een tabel te maken heb je volgende elementen nodig, Geef voor elke element een definitie.:

  • <table>

    • Het HTML <table> element vertegenwoordigt gegevens in tabelvorm, dat wil zeggen informatie die wordt gepresenteerd in een tweedimensionale tabel die bestaat uit rijen en kolommen met cellen die gegevens bevatten.

  • <thead>

    • Het HTML <thead> element definieert een reeks rijen die de kop van de kolommen van de tabel definiÃ«ren.

  • <tbody>

    • Het HTML <tbody> element bevat een set tabelrijen (<tr> -elementen), wat aangeeft dat ze de body van de tabel (<table>) vormen.

  • <tfoot>

    • Het HTML <tfoot> element definieert een reeks rijen die de kolommen van de tabel samenvatten.

  • <th>

    • definieert een cel als koptekst van een groep tabelcellen. De exacte aard van deze groep wordt bepaald door de kenmerken scope en headers.

  • <tr>

    • Het HTML <tr> element definieert een rij cellen in een tabel. De cellen van de rij kunnen vervolgens worden gemaakt met behulp van een combinatie van <td> (datacel) en <th> (header) -elementen.

  • <td>

    • Het HTML <td> element definieert een cel van een tabel die gegevens bevat. Het neemt deel aan het tafelmodel.
