18. Onderzoek welke 4 attributen een link tag kan hebben, leg uit wat deze doen en voeg deze als voorbeeld toe op je website.

Het link element creeert een verband tussen je HTML document en een extern document

<rel>

Specificeerd de relatie tussen twee documenten

<media>

Het mediaattribuut vertelt de browser voor welk type apparaat elk script is ontworpen.U kunt media gebruiken om de browser te vertellen dat u verschillende stylesheets heeft om weer te geven en af ​​te drukken. Op die manier ziet uw bezoeker een versie die is geoptimaliseerd voor hun mobiele apparaat wanneer deze op een klein scherm wordt bekeken, en tegelijkertijd krijgen ze een andere lay-out bij het afdrukken.

<title>

Wijst een naam toe aan verschillende gekoppelde bronnen, zodat gebruikers ertussen kunnen kiezen.

<type>

Beschrijft het mediatype van de gekoppelde bron (MIME type).
