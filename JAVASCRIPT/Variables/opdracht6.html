Hoe herken je dat een waarde String is? Geef ook een voorbeeld.

Een string is wat als tekst geldt. Om aan te duiden dat de variabele een string is, moet u die tussen aanhalingstekens zetten.

Hoe herken je dat een waarde Number is? Geef ook een voorbeeld.

Een nummer. Nummers worden niet omringd door aanhalingstekens .

Hoe herken je dat een waarde BigInt is? Geef ook een voorbeeld.

BigInt. Een BigInt lijkt heel veel op Number. De waarde van BigInt is een getal en eindigt op n. Bijvoorbeeld 3n. De grootste verschil tussen BigInt en Number is dat BigInt oneindig getallen kan opslaan. Dit is niet nodig om te onthouden.

Hoe herken je dat een waarde Symbol is? Geef ook een voorbeeld.

Symbol. Symbol wordt gebruikt om de waarde een unieke key van te maken . Wordt heel weinig gebruikt en is niet nodig om te onthouden. checken met ..typeof x === 'symbol'
